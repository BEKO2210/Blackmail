Auto-Switch — Email Notifications

The simplest way: enter your guardians' emails. If you stop checking in, we automatically email them. No accounts, no apps for your guardians.

Hosted service. Unlike the rest of AEGIS, this feature uses a server to store your interval, your message and your guardians' emails so it can send mail automatically. Your message is stored on the server — keep highly sensitive evidence in the client-side encrypted package and only link to it here.
The hosted service is not configured yet. The operator must deploy the Worker and set the API URL in js/service-config.js.
